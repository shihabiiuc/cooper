CSS has evolved rapidly. Features that once required preprocessors or complex JavaScript are now native.

### New Features

1.  **Nesting**: Native nesting support reduces the need for Sass.
2.  **Container Queries**: Style elements based on their container size, not the viewport.
3.  **Scoped Styles**: `@scope` allows for true component-level styling without naming collisions.
4.  **Trigonometric Functions**: Use `sin()`, `cos()`, and `tan()` directly in CSS.

While we love Tailwind for its utility-first approach, understanding modern CSS allows you to write cleaner, more performant code when you need custom components.
